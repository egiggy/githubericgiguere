A) Getting Started - top

In order to begin installing 8Cells theme. You need the latest version of Wordpress. Here is the download URL. Once you downloaded and set it up, You are ready to go.

Installation

Upon downloading the package, you will have an "8Cells" folder inside the theme package. Upload this folder to your themes folder at: your WordPress installation root directory /wp-content/themes

Log into WordPress administration and click on the Appearance >> Themes tab. After that is done, activate the theme as you would activate any other theme.

Note: You have to set /cache folder of 8Cells writable (chmod 777)


Setup Main Menu

8Cells has 2 menus locations, the menu align next to the logo and the one at the bottom left.

Go to Appearance > Menus then click "Create Menu" name it "Main Menu". Add items to your menu and select Theme location "Main Menu".

Go to Appearance > Menus then click "Create Menu" name it "Bottom Menu". Add items to your menu and select Theme location "Bottom Menu".



For those who are not familiar with Wordpress 3.0 menu below are some tutorials.

Theme Admin Panel

Upon activating the theme, from your WordPress admin panel, you will notice the "8Cells" options tab at the bottom of your WordPress admin panel menu options. Click on it. You will open the admin panel of 8Cells.



Theme Style

You can choose theme style between fixed width and full width, change your logo, favicon and other settings via 8Cells admin > General.


How to change homepage contents?

  1. Go to 8Cells Admin Panel > Homepage.

How to add a homepage slide?

  5. Click on "Publish" to make this slide viewable.

How to add a portfolio page?

  1. Click on "Pages > Add New"
  2. Enter page title
  3. On Page Attributes box, select template as "Portfolio"
  4. Click on "Publish" to make this page viewable.


How to add a portfolio post?

  1. Click on "Add New Portfolio"
  2. Enter title, its content
  3. (Optional) Select the portfolio set
  4. Click on "Set featured image" to upload preview image for this portfolio
  5. Once uploaded click "Use as featured image"
  6. Click on "Publish" to make this post viewable.


You can change portfolio styles, change its sorting and other options via 8Cells admin > Portfolio


How to use portfolio set?

You can use set to organize your portfolio items (similar to category for blog posts)

Once you assign portfolio to set. Go to appearance >menus. Then select which set you want to add to your menu items. If you can see it, click on screen option (top right) and check on sets.



How to add a photo gallery?

  1. Open Galleries > Add New
  2. Enter gallery's name and description
  3. Publish it


How to add a gallery's image?

  1. Open gallery page
  2. Click on Gallery Images > Add New button
  3. Select files to upload (you can use SHIFT or CTRL for multiple images uploading)
  4. Once the upload progress is completed. It will automatically add to gallery


How to add a gallery to main menu?

  1. Go to Appearance > menus
  2. There is a "Galleries" box. Select which gallery you want to add.
  3. Once finish organizing, click on "Save Menu"


If you can't see it, click on screen option (top right) and check on galleries

How to add a blog page?

  1. Click on "Pages > Add New"
  2. Enter page title
  3. On Page Attributes box, select template as "Blog"
  4. Click on "Publish" to make this page viewable.


How to add a blog post?

  1. Click on "Posts > Add New"
  2. Enter title, its content
  3. (Optional) Select the post category
  4. Click on "Set featured image" to upload preview image for this post
  5. Once uploaded click "Use as featured image"
  6. Click on "Publish" to make this post viewable.

You can choose post's featured content from video, images or gallery via "Post Options" on the left side of edit post page.

How to add a contact page?

  1. Click on "Pages > Add New"
  2. Enter page title
  3. On Page Attributes box, select template as "Contact"
  4. Click on "Publish" to make this page viewable.

Then go to 8Cells Admin Panel > General and setup your email address.


How to change site's elements colors?

  1. Go to 8Cells Admin Panel > Colors.
  2. Choose the color from color picker for each element


8Cells custom widgets

However the homepage sidebar is support all Wordpress 3.0 widgets. I created 7 custom widgets to make your website more interesting.

You can also add your sidebar via Admin panel.

To add content to sidebar simple go to Appearance > Widgets and drag item into certain sidebar.


Sidebar widget with shortcode support

You can also add shortcode to sidebar widget, simply select "Text" type for your widget and enter shortcode to it.


Page styles "Full width", "Left Sidebar" or "Right Sidebar"

8Cells has 3 page styles, "Full width", "Left Sidebar" or "Right Sidebar". To select page style. Go to add new page. Fill enter the title field and description. Go to Page Options then select page style box.


Shortcode Generator

8Cells also come with built-in plugin called "Shortcode Generator" which will help you create theme's shortcode from GUI. It will automatically shows in your edit page, post.

Select shortcode type you want to add then enter its contents. Click on "Generate Shortcode" and copy all code inside Shortcode box. Go to your post or page paste the code to its content.


Troubleshooting

8Cells also come with built-in plugin called "Troubleshooting" which will help you solve common issues that might arise from different server settings.

B) CSS Files and Structure - top

I'm using five CSS files in this theme. The first file contains all of the specific stylings for the page. The file is separated into sections using:

	
	/* CSS Reset */

	some code

	/* Global layout */
	
	some code
	
	/* Input layout */
	
	some code


The second file is a styling for jQuery UI element.

The third file is a styling for modal window.

The fourth and fifth file is a styling for HTML5 video.

If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.
